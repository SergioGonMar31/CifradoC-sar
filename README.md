# CifradoCesar
# 🔐 Cifrado César en Java

Este proyecto implementa un **algoritmo de cifrado César** desarrollado en **Java** utilizando el entorno de desarrollo **Eclipse**.

El programa permite **cifrar y descifrar mensajes** desplazando las letras del alfabeto un número determinado de posiciones.

---

## 🛠 Tecnologías utilizadas

- Java
- Eclipse IDE

---

## ⚙️ Instalación y ejecución

Descarga o clona este repositorio y abrelo con Eclipse.
